## Introduction
  由於現在大部分人皆是使用Line作為傳輸訊息之工具, 且以我來說. 比較不愛開電腦或網頁, 因此我使用Line作為一個小幫手，幫助我達到一些目的。

  2. 翻譯
  3. 即將上映之電影
  4. 即時新聞
  5. 及時股價
